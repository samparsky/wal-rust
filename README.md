# wal-rust

A simple and fast write ahead log for Rust.

## Features

- High durability
- Fast writes
- Low memory footprint
- Monotonic indexes
- Log truncation from front or back.

## Getting Started

Add this lib as a dependency

```toml
[dependencies]
wal-rust = "0.1"
```
